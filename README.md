# O que é Sabores da Terra?
É uma plataforma composta por uma loja online e um bot do Telegram para venda de produtos orgânicos regionais.

## Sobre o projeto

O Sabores da Terra é uma aplicação desenvolvida para apoiar produtores locais na organização e divulgação de seus produtos.

A plataforma permite:

- Cadastro de produtos

- Visualização de itens disponíveis

- Organização de informações sobre alimentos

- Divulgação de produtos orgânicos ou artesanais

- Facilitar o contato entre produtores e consumidores.

A proposta é valorizar produção local, e alimentos naturais.

Veja a documentação completa [aqui](https://guilopes15.github.io/sabores-da-terra/).
